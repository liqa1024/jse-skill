# IHas{Type}SetIterator

> `jse.code.iterator`

**核心职能**：6 种 `@FunctionalInterface`，`extends IHas{Type}SetOnlyIterator`，返回读写型迭代器。

```java
// IHasDoubleSetIterator (extends IHasDoubleSetOnlyIterator)
IDoubleSetIterator setIterator()
```

其余 5 种：`IHasBooleanSetIterator`, `IHasIntSetIterator`, `IHasLongSetIterator`, `IHasFloatSetIterator`, `IHasComplexDoubleSetIterator`。
