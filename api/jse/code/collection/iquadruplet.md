# IQuadruplet / Quadruplet

> `jse.code.collection.IQuadruplet` / `jse.code.collection.Quadruplet`

**核心职能**：`IQuadruplet<A,B,C,D>` 为只读四元组接口，`Quadruplet<A,B,C,D>` 为可变实现。

```java
// IQuadruplet<A,B,C,D>
A getA() / B getB() / C getC() / D getD()

// Quadruplet<A,B,C,D>
A mA / B mB / C mC / D mD
Quadruplet(A, B, C, D)
```
