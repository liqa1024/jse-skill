# IDoublePair / DoublePair

> `jse.code.collection.IDoublePair` / `jse.code.collection.DoublePair`

**核心职能**：`IDoublePair` 为 double 专用只读二元组，`DoublePair` 为可变实现（`mFirst`/`mSecond` 字段）。

```java
// IDoublePair
double first() / double second()

// DoublePair
double mFirst / double mSecond
DoublePair(double, double)
```
