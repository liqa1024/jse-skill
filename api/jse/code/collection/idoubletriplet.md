# IDoubleTriplet / DoubleTriplet

> `jse.code.collection.IDoubleTriplet` / `jse.code.collection.DoubleTriplet`

**核心职能**：`IDoubleTriplet` 为 double 专用只读三元组，`DoubleTriplet` 为可变实现。

```java
// IDoubleTriplet
double getA() / double getB() / double getC()

// DoubleTriplet
double mA / double mB / double mC
DoubleTriplet(double, double, double)
```
