# DoubleTriplet

> `jse.code.collection.DoubleTriplet`

**核心职能**：`implements IDoubleTriplet`。double 专用三元组的可变实现，包含 `mA` / `mB` / `mC` 公共字段。

```java
// === 字段 ===
double mA
double mB
double mC

// === 构造器 ===
DoubleTriplet(double, double, double)

// === IDoubleTriplet 接口实现 ===
double a()
double b()
double c()

// === 对象方法 ===
boolean equals(Object)
int hashCode()
String toString()
```
