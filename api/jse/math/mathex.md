# MathEX

> `jse.math.MathEX`

**核心职能**：静态工具类。包含数学常量和一个复数虚数单位常量 `i1`。下设 5 个静态内部类。

```java
// === 常量 ===
static final double PI, E                           // 圆周率, 自然常数
static final double SQRT2, SQRT2_INV                // √2, 1/√2
static final double SQRT3, SQRT3_INV, SQRT3DIV2     // √3, 1/√3, √(3/2)
static final IComplexDouble i1                      // 虚数单位 (0+1i)
static final int SH_LARGEST_L = 200                 // 球谐函数最大 l
```
