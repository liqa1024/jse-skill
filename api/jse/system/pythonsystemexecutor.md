# PythonSystemExecutor

> `jse.system.PythonSystemExecutor`

**核心职能**：final class extends AbstractSystemExecutor。Python 脚本执行器，支持 python / python3 自动选择。

```java
PythonSystemExecutor(boolean aUsePython3)  // true→"python3", false→"python"
PythonSystemExecutor()                      // → Python(false)
```
