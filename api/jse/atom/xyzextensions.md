# XYZExtensions

> `jse.atom.XYZExtensions`

**核心职能**：为 Groovy 提供 `Number + IXYZ` 等左侧扩展运算（类似 Groovy `@Extension` 风格）。纯静态方法类。

```java
static XYZ plus(Number, IXYZ)       // Number + xyz
static XYZ minus(Number, IXYZ)      // Number - xyz
static XYZ multiply(Number, IXYZ)   // Number * xyz
static XYZ div(Number, IXYZ)        // Number / xyz
```
